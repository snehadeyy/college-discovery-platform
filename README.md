# College Discovery Platform

A full-stack college discovery and comparison platform inspired by Careers360 and Collegedunia.
Users can search, filter, explore, and compare colleges to make better academic decisions.

# Tech Stack
- Node.js
- Express.js
- React.js + Tailwind CSS
- Postgre SQL
